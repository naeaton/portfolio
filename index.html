Project Work

Loan Approval Model

Using SQL and Python's pandas library, I cleaned and analyzed loan data extracted from Microsoft Access. After preprocessing, I trained a predictive model to determine loan approval likelihood based on applicant characteristics. Key features influencing approval were identified, offering insights into lending decisions. The model achieved an accuracy rate of 95%.

LC2K Assembler

I developed an assembler for LC2K assembly files. It processes the code, creating object files for each, and then links them into a unified executable. This simplifies the conversion of LC2K assembly code into a single, runnable program.

Banking Backend

I developed a banking backend system that securely manages user balances, login credentials, and tracks login attempts. The system features a query-like interface enabling users to simulate transactions and other banking operations, providing a realistic banking experience in a simulated environment.

Real-Time Room Occupancy Tracker

This project developed a real-time room occupancy tracking system using laser sensors, a Raspberry Pi, Python scripting, and web development technologies. Laser sensors, connected via a breadboard to a Raspberry Pi, accurately detected movement at entry points. Python scripts processed sensor data to maintain a 95% accurate count of people entering and exiting the room. An HTML/CSS web application provided users with an intuitive interface to access real-time occupancy information. This system offers versatile applications in various settings like offices and public venues, providing reliable and accessible occupancy monitoring.

Travel Information Website

I developed a website using HTML, CSS, and JavaScript showcasing travel information for specific destinations. Each page highlights details about different places, offering visitors a visually appealing and interactive experience to explore various travel spots.

3D Maze Solver

I developed a solution for navigating 3D mazes using Breadth-First Search (BFS) and Depth-First Search (DFS) algorithms to find optimal routes. This program efficiently explores maze structures, providing users with the shortest or deepest paths from start to finish. The implementation showcases algorithmic proficiency and practical applications in solving complex spatial puzzles.

Football Statistics

Using R, I analyzed scores from an Excel dataset for the University of Michigan and Ohio State football teams. Statistical tests were conducted to determine if one team outperformed the other in the past year. The results showed no statistically significant difference in performance between the two rivals, shedding light on the parity in their recent performances.

Web Server for Office Hours Queue

This project involved developing a server that processed JSON input requests from clients. Upon receiving an HTTP request containing JSON data, the server deserialized it, modified an internal data structure accordingly, created a response, serialized it back to JSON format, and sent it back to the client. This streamlined process facilitated efficient communication between clients and the server, showcasing effective handling of JSON-formatted data for web-based applications.
